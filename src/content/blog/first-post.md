# はじめに

この度、Astroを使用して個人のホームページおよびブログを新しく立ち上げました。
デザインはダークテーマを基調とし、シンプルで読みやすい構成を目指しています。

## なぜAstroを選んだのか

Astroは静的サイトジェネレーターとして非常に優れています。主な理由は以下の通りです：

1. **パフォーマンス**: デフォルトでJavaScriptを出力しないため、非常に高速です。
2. **コンポーネント指向**: React, Vue, Svelteなどのフレームワークを混在させることができます。
3. **コンテンツコレクション**: MarkdownやMDXの管理が型安全に行えます。

```javascript
// コンテンツコレクションの例
import { defineCollection, z } from 'astro:content';

const blog = defineCollection({
  schema: z.object({
    title: z.string(),
    pubDate: z.date(),
  })
});
```
